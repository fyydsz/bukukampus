## 1. Variabel â€” Tempat Menyimpan Data
Variabel adalah **wadah** untuk menyimpan nilai.  
Cara membuat variabel: **nama_variabel â†’ `=` â†’ nilai**.

```python filename="variabel.py"
# Membuat variabel
nama = "Raffi"       # Teks (String)
umur = 21            # Angka bulat (Integer)
tinggi = 170.5       # Angka desimal (Float)
is_pelajar = True    # Benar/Salah (Boolean)

# Mencetak isi variabel
print(nama)
print(umur)
```

Python langsung mengenali tipe datanya, jadi kamu **tidak perlu** menyebutkan tipe secara manual.

## 2. Indentasi â€” Spasi yang Wajib Konsisten
Python memakai **indentasi** (spasi/tab di awal baris) untuk menentukan blok kode.  
Kalau spasinya salah â†’ program error.

```python filename="indentasi.py"

    # Baris ini menjorok ke dalam
    print("Sepuluh lebih besar dari lima")
    print("Ini masih di dalam blok if")

```

## 3. Komentar â€” Catatan untuk Manusia
Komentar adalah teks yang **tidak dijalankan** oleh komputer.

```python filename="komentar.py"

```

## 4. Tipe Data Dasar
Tipe data = jenis nilai di dalam variabel.

| Tipe Data            | Contoh               | Keterangan                           |
| :------------------- | :------------------- | :----------------------------------- |
| **String** (`str`)   | `"Halo"`, `'Python'` | Teks (harus dalam tanda kutip)       |
| **Integer** (`int`)  | `10`, `-5`, `0`      | Bilangan bulat                       |
| **Float** (`float`)  | `3.14`, `2.5`        | Bilangan desimal                     |
| **Boolean** (`bool`) | `True`, `False`      | Nilai benar/salah                    |

Contoh mengecek tipe data:

```python filename="cek_tipe.py"
nilai = 100.5
print(type(nilai))
# Output: <class 'float'>
```
